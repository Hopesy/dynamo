# Python Dynamo开发环境搭建

推荐使用VS CODE搭建开发环境。revit APi相关包如下：

「ironpython-stubs-master」 链接：https://www.aliyundrive.com/s/xiSVkirXAXbopen in new window

前言：Dynamo中的Python Script缺少语法检查和自动补充。因此我们需要在外部开发环境中进行IronPython的配置，提高开发效率。详见https://github.com/gtalarico/ironpython-stubsopen in new window，去下载ironpython-stubs

# 1.Visual Studio

1.首先下载Visual Studio，推荐2019版本。详见https://zhuanlan.zhihu.com/p/94998894open in new window

注意: Dynamo2.7版本依赖于IcronPython(Python2.x在.net平台的版本)

2.新建一个Python项目

3.将下载解压的Ironpython-stubs-master中的\release\stubs.min添加到搜索路径,此时编辑Python Script就有代码提示了，以后的项目均可以在此项目下新建PY文件测试完成后复制进Dynamo节点中即可

# 2.Atom

ATom自从被微软收购之后，项目已经烂了，基本没人用了，都改为使用VS CODE了。

# 1.下载安装Atom编辑器，详见https://atom.io/open in new window

# 2.搭建Python开发环境（首先保证已经安装Python）

（1）File -> Settings ->Packages 检查Python语法支持

(2)打开Git Bash，运行以下指令安装python-language-server

pip3 install python-language-server[all]

(3)安装支持各种语言IDE的UI界面，+Install里搜索atom-ide-ui

(4)同上，安装安装ide-python和运行atom-python-run,代码补全autocomplete-python，语法检查linter-flake8

# 3.Atom编辑器其他常用插件

(1)simplified-chinese-menu:中文汉化 (2)Atom-file-icons:文件图标

# 3.VS CODE（推荐使用）

下载安装vs code及Python拓展支持

  • 官网下载安装VS CODE https://code.visualstudio.com/open in new window
  • Ctrl+Shift+X，进入插件管理页面，搜索Py安装Python和Python Extensions pack

新建项目： 进入本地新建文件夹例如:pytest,切换到VS CODE并打开该文件夹

选择解析器： Ctrl+Shift+P，进入控制台，搜索并点击Python Select Interpreter选择一个Python版本即可，自动在本项目文件夹下生成配置文件setting.json（生成的文件默认会配置Python的解析器,此配置文件只对此项目有效)。目前遇到的问题是插件下载太慢，导致项目配置文件生成失败。

全局配置： 如果上一步没有生成setting.json，那么我们打开全局设置进行配置Revit Api代码补全，也是可以的，没任何问题。

  • 打开用户设置

  • 将下载下来的Revit 二次开发的包路径，添加在setting.json。

"python.autoComplete.extraPaths": [
        "C:\\ironpython-stubs-master\\release\\stubs.min",#Python Script代码自动补全，替换成自己的
        "C:\\Python27amd64\\Scripts" #Python2.x路径，替换成自己的
    ]

配置完成，可以在此项目文件下新建py文件进行开发

语法检查插件安装(这一步不做也可以正常使用)：使用pip安装其他的包(IcronPython基于Python2.x，所以使用Python2环境)。

打开cmd命令行安装

pip2 install flake8 #检查编写代码时的不规范的地方和语法错误。(我同时安装了python2和python3为了区分将pip改名了)
pip2 install yapf  #代码格式化工具，可以一键美化代码。

配置插件

对flake8和yapf进行配置打开user的setting.json加上以下

"python.linting.flake8Enabled": true,
   "python.formatting.provider": "yapf",
   "python.linting.flake8Args": ["--max-line-length=248"], # 设置单行最长字符限制
   "python.linting.pylintEnabled": false # 关闭pylint工具

其他

  • PYlance有时候插件无法识别Python2.x，需要卸载
  • 用户的配置文件(WorkSpace，你项目文件夹下的settings.json)会覆盖掉默认的配置文件(User,你在左下角设置setting那里点开的)
