# Dynamo连接数据库

本案例通过Python Script连接MongoDb数据库，获取需要标高的名称和高度数据

# 1. 安装高版本Dynamo

由于低版本Dynamo使用IronPython，不支持PyMongo库。因此首先需要安装Revit 2020,默认继承了Dynamo 2.10。

# 2. 下载安装MongoDb

先在本地电脑安装案例需要用到的MongoDb数据库。MongoDB将数据存储为一个文档，数据结构由键值(key=>value)对组成,类似Python中的字典。

Windows直接去官网点击下载。 双击安装包进行安装

为了方便启动，我们将软件bin目录放在系统环境变量中,这样无论在任何目录启动Bash或者cmd都可以直接使用mongo命令启动数据库。

MongoDB 默认将数据库文件存储在安装路径下的data目录下，日志默认存放在log目录下。

为了避免文件混乱。我们一般新建一个db目录存放数据库，然后修改配置文件指向此路径。（不想改的话这一步不做也行）

**注意：**我们安装的时候勾选了注册为服务，只有注册了服务才能用net start mongodb的方式启动，否则只能在命令行启动mongod -dbpath "D:\MongoDb\data\db"可以跟很多启动参数。安装时也可以不勾选，手动注册服务命令。

#这一步不需要
mongod --dbpath "D:\MongoDb\db"  -serviceName "mongodb" -serviceDisplayName "mongodb" -install

启动Mongodb服务（不区分大小写）。

net start mongodb
//不指定端口的话默认是http://127.0.0.1:27017

启动之后就能使用mongo命令进入数据库了，默认已经建了个表test

如果我们希望操作起来更简单，就需要安装可视化界面。

下载后任意路径解压双击MongoDBCompass即可使用。

点击fill in connection填写网址端口connect即可连接。

# 3. 准备模拟数据

本次案例新建数据库hopesy,新建合集level，插入几条模拟数据。

# 4. 安装 pymongo库

使用pip安装pymongo库。（pip是Python的包管理工具，安装之后就可以在终端中使用）。

pip install pymongo

# 5. Python Script

打开Dynamo ,将我们在vs code调试好的代码粘贴过去。案例比较简单就不再介绍了，注意把下载的pymongo包的地址加载进来就行。我们拿pip下载的包都在python安装目录的site-packages路径下

import sys
#加载pymongo包，首先要安装python后再pip install mongodb
#路径替换成自己的
sys.path.append('D:\python39\lib\site-packages')
from pymongo import  MongoClient
#连接数据库
myclient = MongoClient("mongodb://localhost:27017/")
dblist = myclient.list_database_names()
if "hopesy" in dblist:
    mydb = myclient["hopesy"] 
else:
  #print可以改为异常捕获啊，警示对话框之类的
  print("hopesy数据库未建立")
mycollection = mydb.list_collection_names()
if "level" in mycollection :
  levelCollevtion = mydb["level"]
else:
  print("level合集未建立")
##按照高度对数据排序，不做也行
datas = levelCollevtion.find().sort("elevation")
name =[]
ele = []
ce = []
for da in datas :
  ele.append(da["elevation"])
  name.append(da["name"])
  ce.append(da["注释"])  
#输出数据
OUT = name,ele,ce
