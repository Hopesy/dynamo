# 前言

个人建议，多看看官方文档，再找几个案例，没必要花钱买课程。遇到问题先百度，推荐先看姑娘课堂的视频入门，至于进阶的Python 相关的，先过一遍Python Primer,再多研究Revit API就行了。

公众号推荐牛侃、九哥，都是干货。当然关注我也可以，不过我写的东西不太正经，微信里面直接搜索hopesy，或者扫码关注。

# 1.Dynamo官方下载

Dynamo官方网站，下载各种安装包；使用流程；节点介绍;详细案例。详见https://dynamobim.org/open in new window

# 2.Dynamo Primer

(Dynamo官方使用教程)，内容系统且支持简体中文，无需登录，可以在线观看也可下载PDF,详见https://primer.dynamobim.org/zh-cn/index.htmlopen in new window

# 3.Dynamo Dictionary

(官方节点介绍)，可以直接搜索节点且可下载样例全部内置节点，详见:https://dictionary.dynamobim.com/#/open in new window
